# Instagram Clone

A React Native application that replicates core Instagram features built with Expo.

## ✨ Features
- 🔐 User authentication with Firebase
- 📸 Photo sharing with camera/gallery upload
- 🎥 Stories with 24-hour expiration
- 💬 Direct messaging with real-time updates
- 🔍 Explore feed with personalized recommendations
- ❤️ Like and comment functionality
- 🛒 Shopping features (product tags)

## 🚀 Installation
```bash
git clone https://github.com/itsaahsan/Instagramclone.git
cd Instagramclone
npm install
expo start
```

## ⚙️ Requirements
- Node.js v16+
- Expo CLI
- Android/iOS simulator or physical device
- Firebase project setup (for authentication)

## 📱 Screens
- Home Feed
- Explore
- Camera
- Reels
- Profile
- Notifications

## 🔧 Technologies Used
- React Native
- Expo
- Firebase
- Redux
- React Navigation
